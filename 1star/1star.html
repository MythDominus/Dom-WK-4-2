Your sketch:

Instructions

You will create an interactive animation as shown below.

  1. gameObjects.js contains a CollisionRect class. Familiarise yourself with the public methods.
  2. In gameObjects.js, create another class called Ball, which inherits from CollisionRect. Ball should have three attributes representing the ball's position and speed on the y axis. When the ball is created, it should be positioned in the middle of the top of the screen. Set the speed to 5.
  3. Add a draw() and move() methods to the Ball. draw() should draw the ball (you can decide its size and colour) and move() should use the speed attribute to move it down the y axis.
  4. Create another class called Catcher that inherits from CollisionRect. The catcher will need location attributes but not speed.
  5. Add a draw() method, which should draw a rectangle (you choose the size and colour) at the catcher's coordinates.
  6. In sketch.js, use your new classes to create the interactive animation shown below. The ball should drop from the top of the canvas and the catcher should move with the user's mouse. If the ball hits the catcher it should stop moving. Pressing "s" should reset the ball's position with a random x coordinate.

Expected output:

In your Ball and Catcher classes, don't forget to use the parent class methods to set and update the CollisionRect position and dimensions. You will also need to call super() in both constructors even though the parent constructor is empty.
